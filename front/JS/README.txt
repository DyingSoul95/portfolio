project1 - При входе на сайт пользователя должно встречать окно с запросом «Введите ваше имя». После того как пользователь ввел своё имя, требуется уточнить, верно ли он ввёл его. И если пользователь нажал кнопку подтверждения, то 	должно появиться окно приветствия, в котором отобразится: «Добро пожаловать, <введенное_имя>».

project2 - Добавьте на вашу страницу фрейм. Под фреймом должно располагаться поле для ввода и кнопка. При нажатии на эту кнопку внутри фрейма должен открываться google с тем запросом, который был указан в поле ввода. Если в поле ввода 	ничего не указано, то должна быть открыта стартовая страница Google.com

project3 - В мебельном магазине имеется постоянно обновляющаяся таблица с информацией о товарах. Владельцу необходимо знать о количестве товара, который у него продается. Ваша задача написать функцию count(), в которой подсчитывается 	количество продукции, имеющейся в магазине. Функция должна быть привязана к кнопке и выводить на экран строку «Уникальных товаров в магазине: <count>» при помощи метода alert. 

project4 - На вашей странице имеются 2 кнопки – «Показать текст», «Скрыть текст». Ваша задача описать 2 функции – show() и hide(), которые по нажатию на соответствующие кнопки показывали или скрывали текст на странице. Функция show() 	должна записывать в каждый div строку в формате «Строка <номер_строки>» (Нумерация начинается с 1). Функция hide() должна скрывать весь текст на странице, оставляя видимыми лишь кнопки.

project5 - На сайте выставочного центра нужна форма регистрации участников. Любой желающий может ввести туда свои инициалы и добавить себя в список. Ваша задача описать функцию addMember(), в которой получать введенное значение из 	текстового поля и добавлять его в отдельный div.

project6 - Приведите определение javascript функции sortUl(), которая выполняет сортировку списков (ul), размещенных на странице, по алфавиту. При сортировке необходимо учитывать, что списки могут быть многоуровневыми. Необходимо 	реализовать алгоритм таким образом, чтобы сортировка происходила по уровням, с учетом иерархии элементов.

project7 - На сайте университета выставлен список абитуриентов. Для удобства чтения информации приемная комиссия просит вас добавить подсветку строк таблицы, при клике на них курсором. Ваша задача описать функцию, которая будет выделять 	всю строку таблицы при клике на любую из ячеек этой строки
