Описание заданий, по которым я делал данные странички

project1 - В интернет-магазине одежды требуется немного изменить вид чека. Заголовки должны отображаться по центру страницы. Текст «Спасибо за покупки!» должен быть серого цвета(#AFAFAF) и размером 28px. Внутри абзацев текст должен быть 	размером 24px. Текст в таблице следует выровнять по центру.

project2 - На сайте завода ежемесячно публикуются списки лучших сотрудников, которые отличились своей продуктивностью. Ваша задача подчеркнуть некоторые значимые фрагменты в тексте. На странице должны быть реализованы следующие 	требования:

		Размер текста внутри абзацев – 24px
		Если текст, выделенный жирным, следует за обычным текстом, он должен быть зеленого цвета(color: green;).
		Текст после списка записывается курсивом
		Текст в теге <p> следующий за тегом <div> выделяется красным цветом

project3 - Ремонтно-строительная компания делает свой сайт-визитку. На главной странице они хотят разместить список с услугами и ценами. Все элементы списка должны быть прописаны курсивом. Цена на услуги должна быть подчеркнута, иметь 	цвет #1090EF и не должна быть курсивной. Размер текста на странице – 20px.

project4 - Предыдущий разработчик создал форму для регистрации на сайте, однако с CSS у него возникли проблемы. Твоя задача доделать внешний вид страницы регистрации согласно требованиям:
	
		Все элементы страницы должны находиться по центру
		Размер текста на странице -24px
		Отступ между абзацами – 15px
		Размер формы для ввода – 300х30px
		Размер текста внутри формы ввода – 24px
		Размер кнопки регистрации -400x40px
		Размер текста на кнопке – 24px

project5 - На сайте представлена информация об одном медиаплеере для того, чтобы пользователи могли ознакомиться с функциями и особенностями данной программы. Названия компаний и программных продуктов должно быть как-то выделено в 	тексте. Ваша задача состоит в том чтобы:

		Установить темно-синий цвет(#3000a8) для названий компаний и ПО, находящихся внутри тега <em>
		Подчеркнуть ранее выделенные названия
		Установить светло-розовый цвет(#fef0f1) для любых элементов внутри контейнера <div>

project6 - На сайте ресторана представлено меню с выбором блюд из разных кухонь. Ваша задача, используя селекторы псевдоклассов, добавить цвета к четным и нечетным строкам таблицы. Строки с нечетными номерами должны иметь светло-зеленый 	цвет (#ccfccc), а строки с четными номерами – светло-голубой(#d3e2f4). При наведении курсора на ячейку в таблице, она должна окрашиваться в светло-красный цвет(#f56e6e), подчеркиваться и становиться жирным.

project7 - Строительный магазин «Отделочник» добавил на свой сайт страницу с каталогом товаров. Предыдущий разработчик настроил дизайн страницы, но сделал это недостаточно оптимизировано. Ваша задача упростить данный код, используя 	группировку селекторов. В качестве ответа приведите css-файл с исправленными стилями.
